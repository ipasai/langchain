🤖 LangChain 多模型應用

這是一個整合 AI 對話、資料庫管理與英文學習的多功能 Web App，讓使用者可以在同一個介面中完成模型互動、資料查詢與單字管理。

核心目標

  • 支援多模型 AI 對話
  • 提供資料庫管理與查詢
  • 輔助英文學習與單字複習

支援平台

OllamaOpenAIGoogle Gemini
SQLiteMySQLPostgreSQL MariaDBRedisMongoDB
由 LangChain + Streamlit 構建，支援本機記憶與資料持久化

💬 AI 對話功能

  • 可在側邊欄切換模型提供者
  • 可調整溫度參數，控制輸出風格
  • 保留對話歷史，方便後續回顧
  • 支援清除歷史與重新開始新對話
  • 支援自然語言提問
  • 可直接串接不同 LLM 服務
  • 適合做問答、翻譯、摘要與日常助理

🗄️ 資料庫管理功能

  • 支援 SQLite、MySQL、PostgreSQL、MariaDB、Redis、MongoDB
  • 可輸入連線設定並進行測試連線
  • 自動分析資料表結構、欄位與關聯
  • 可預覽表格內容與資料行數
  • 支援 SQL 查詢執行
  • 支援資料列 CRUD 操作
  • 支援單筆刪除與批次編輯
  • 提供更漂亮的資料表編輯介面與 CSV 下載

📘 英文學習功能

  • 可新增英文單字與中文翻譯
  • 可附加例句並儲存到本機資料庫
  • 支援依照分類管理單字
  • 可在分類單字清單中查看例句與刪除單筆資料
  • 提供單字卡片複習模式
  • 支援收藏與今日複習紀錄
  • 提供 TOEFL 風格測驗與學習成效統計
  • 可追蹤最近準確率與每日趨勢

🔍 字典式查詢與例句生成

  • 先查詢本機已儲存的翻譯資料
  • 若沒有找到，才會請模型生成翻譯內容
  • 查詢結果可直接填入儲存欄位
  • 支援新增與合併多個例句
  • 支援生成多個例句並追加到原有例句
  • 儲存時會把完整例句群一起寫入資料庫
  • 後續查詢與顯示時可完整呈現

🚀 開始使用方式

  1. 啟動虛擬環境並安裝相依套件
  2. 執行主程式：python main.py
  3. 選擇 AI 對話、資料庫管理或英文學習頁籤
  4. 依需求設定模型、連線與學習資料
